# Blender Addon : Randomize Transform

Randomize Transform of selected objects

![Demo](.media/demo.gif)

## ⚙️ Installation

1. Download latest version from [Release](https://github.com/Ni-g-3l/blender-randomize-transform/releases)
2. Install it with Blender Addon Manager
